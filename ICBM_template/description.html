Project Description


The Idaho Department of Health and Welfare is looking to replace its current time and cost tracking system, Mariner. With that being the case, Don Moreaux came to the University of Idaho to have this sytem replaced.

The purpose of ICBM is to create an intuitive and stable web-based Time, Accounting, and Reporting System. TARS will be developed on the Microsoft IIS6 platform, and will utilize the MVC application specifically.

Our project sponsors are:
  • Don Moreaux - Primary Stakeholder
  • Marj Sanderson - Point of Contact
  • The Idaho Department of Health and Welfare

The goal of the first semester development team is to have a working prototype of the software complete by the end of the semester. Since we are working in such a constricted time-frame, our final goals are as follows:
  • IIS7 MVC Architecture Completed and Running
  • Basic Site Design and Interface Completed; up to and including all CSS, Javascript, and C# needed to fulfill this expectation.
  • Active Directory Interface Completed
  • I-Time Interface Completed
  • MySQL Database Schemas Completed and Working
The goal of the second semester development team is to finish the rest of the requirements and deliver a final product. These requirements include:
  • Changing the infrastruture to work with IIS6 instead of IIS7
  • Updating our connection class and connection strings to work with Windows Server 2003
  • Extensive testing
  • Completing the rest of the back-end functionality
  • All of the user interface components
These summarized specifications are defined more exactly in the Requirements Document

In addition, a public GitHub repo contains all our work. GitHub.
